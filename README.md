# javaScript-design-patterns
 javaScript-design-patterns

## 六大原则

1. 单一职责
2. 里氏替换
3. 依赖倒转
4. 接口隔离
5. 最少知识 (迪米特法则)
6. 开发封闭

## 分类

### 三大类

创建型模式 五种: 工厂方法模式 抽象工厂模式 单例模式 建造者模式 原型模式
结构型模式 七种: 适配器模式 装饰器模式 代理模式 外观模式 桥接模式 组合模式 享元模式
行为模式 共十一种: 策略模式 模版方法模式 观察者模式 迭代子模式 责任链模式 命名模式 备忘录模式 状态模式 访问者模式 中介者模式 解释器模式

并发型模式和线程池模式
